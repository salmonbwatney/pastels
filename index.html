Icons

Material Flat supports the full range of icons from Google's Material Design icon collection as well as the Font-Awesome icon collection.

It's easy to incorporate them into the webpage. Here are a couple of examples:

<i class="material-icons">face</i> face

<i class="fa fa-transgender"></i>

To use Google's Material Icons put <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet"> before the end of the <head> tag.

To use the Font Awesome Icons put <link href="https://maxcdn.bootstrapcdn.com/font-awesome/latest/css/font-awesome.min.css" rel="stylesheet"> before the end of the <head> tag.
